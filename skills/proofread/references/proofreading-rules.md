# 교정·교열 규정 (proofreading rules)

한국어 어문 규범에 따라 **명백한 오류**만 바로잡는 기준이다. 근거는 한글 맞춤법, 표준어 규정,
외래어 표기법, 표준국어대사전(국립국어원). 문체·번역투·격식은 이 스킬의 범위가 아니다
(→ humanize / translate-polish / restyle). 여기서는 **틀린 것만** 고친다.

## 분류 체계

| 코드 | 분류        | 다루는 것                                      |
| ---- | ----------- | ---------------------------------------------- |
| 맞   | 맞춤법      | 한글 맞춤법(되/돼, 안/않, 율/률, 사이시옷 등)  |
| 띄   | 띄어쓰기    | 의존명사·보조용언·단위·조사·접사 띄어쓰기      |
| 표   | 표준어      | 비표준어 → 표준어(바램→바람 등)                |
| 외   | 외래어 표기 | 외래어 표기법(메세지→메시지 등)                |
| 부   | 문장부호    | 마침표·쉼표·따옴표·가운뎃점·줄임표 규정        |
| 문   | 문법·비문   | 주술 호응, 조사 오류, 중복 표현 등 명백한 비문 |

## 1. 맞춤법 (맞)

자주 틀리는 핵심 항목. 의미·문맥으로 판별한다.

| 오류             | 교정                | 판별 기준                                          |
| ---------------- | ------------------- | -------------------------------------------------- |
| 되/돼            | 되어=돼             | '하'로 바꿔 말 되면 '되', '해'로 되면 '돼'         |
| 안/않            | 안(부사)/않(용언)   | '아니'면 '안', '아니하'면 '않'                     |
| -데/-대          | 데(경험)/대(전언)   | 직접 경험은 '-더라'=데, 남의 말 전달은 '-다고'=대  |
| -로서/-로써      | 자격=로서/수단=로써 | 자격·지위 '로서', 도구·수단·시간 '로써'            |
| -든지/-던지      | 선택=든지/과거=던지 | 선택·나열 '든지', 과거 회상 '던지'                 |
| -율/-률          | 모음·ㄴ받침=율      | 모음·'ㄴ'받침 뒤 '율'(비율), 그 외 '률'(확률)      |
| 왠/웬            | 왠지/웬일           | '왠지'만 '왠', 나머지는 '웬'(웬일·웬만·웬 떡)      |
| 며칠             | 며칠(O)             | '몇일'은 비표준, 항상 '며칠'                       |
| 사이시옷         | 등굣길·예삿일       | 합성어 발음 [ㄷ/된소리]·뒤 ㄴㅁ 첨가 시(규정 확인) |
| 띄어/띠어, 채/체 | 문맥별              | '채'(상태 유지)/'체'(척); '-째'(접사)              |

> 두음법칙·사이시옷은 예외가 많다. 확신이 서지 않으면 **표준국어대사전을 기준으로 보수적으로** 판단하고,
> 애매하면 교정하지 말고 표시만 한다(과교정 금지).

## 2. 띄어쓰기 (띄)

| 규칙                          | 예 (틀림 → 맞음)                                   |
| ----------------------------- | -------------------------------------------------- |
| 의존명사는 띄움               | 할수있다 → 할 수 있다 / 먹은지 → 먹은 지           |
| 조사는 붙임                   | 너 마저 → 너마저 / 학교 에서 → 학교에서            |
| 단위명사는 띄움               | 한개 → 한 개 / 십이년 → 십이 년 (순서·연월일 예외) |
| 보조용언(원칙 띄움/허용 붙임) | 도와주다(허용)·도와 주다(원칙)                     |
| 접사 '-들/-뿐/-만/-째'는 붙임 | 사람 들 → 사람들 / 이틀 째 → 이틀째                |
| 성+이름은 붙이고 호칭은 띄움  | 홍 길동 → 홍길동 / 홍길동씨 → 홍길동 씨            |
| '안 되다/안되다' 구분         | 일이 안 되다(부정)/공부가 안되다(딱하다) — 문맥    |

> 보조용언·합성어 띄어쓰기는 '원칙/허용'이 공존한다. **한 문서 안에서 일관**되게 통일하는 것을 우선한다.

## 3. 표준어 (표)

| 비표준 → 표준                                                         |
| --------------------------------------------------------------------- |
| 바램→바람, 설겆이→설거지, 금새→금세, 어떻해→어떡해/어떻게, 안되→안 돼 |
| 단언컨데→단언컨대, 폭팔→폭발, 흐리멍텅→흐리멍덩, 일일히→일일이        |
| 통채로→통째로, 역활→역할, 곱배기→곱빼기, 뇌졸증→뇌졸중                |
| 닥달→닦달, 우뢰→우레, 미류나무→미루나무, 께름직→꺼림칙/께름칙         |

## 4. 외래어 표기 (외)

| 틀림 → 맞음                                                         |
| ------------------------------------------------------------------- |
| 메세지→메시지, 컨텐츠→콘텐츠, 악세서리→액세서리, 초콜렛→초콜릿      |
| 쥬스→주스, 비스켓→비스킷, 로보트→로봇, 화이팅→파이팅, 후라이→프라이 |
| 까페→카페, 빵꾸→펑크, 스케쥴→스케줄, 레크레이션→레크리에이션        |
| 플랫폼(O), 앱(O), 데이터(O), 프로포즈→프러포즈                      |

> 외래어 표기법은 원지음·관용을 절충한다. 굳어진 관용 표기(예: 라디오)는 그대로. 신어·고유명사는
> 국립국어원 외래어 표기 용례를 우선한다.

## 5. 문장부호 (부)

- **마침표(.)** — 평서·명령·청유문 끝. 제목·표제어엔 안 씀. 아라비아 숫자 연월일에 사용 가능(2026. 6. 26.).
- **쉼표(,)** — 한국어는 본래 드묾. 연결어미·접속부사마다 찍지 않는다(번역투·AI 티는 humanize 영역).
- **가운뎃점(·)** — 짝·열거의 묶음(빨강·파랑·노랑). 빗금(/)과 혼용 주의.
- **따옴표** — 직접 인용 큰따옴표(""), 강조·생각·재인용 작은따옴표(''). 곧은따옴표("")는 굽은따옴표로 통일 권장.
- **줄임표(……)** — 가운데점 여섯(또는 셋). 마침표 세 개(...)는 비규범, 통일 권장.
- **붙임표·물결표(~)** — 범위(3~5일). 물결 대신 '에서'로 풀 수도 있음.

## 6. 문법·비문 (문)

**명백한 비문만** 고친다. 번역투·어색함 수준은 translate-polish/humanize로 넘긴다.

| 유형               | 예 (비문 → 교정)                                       |
| ------------------ | ------------------------------------------------------ |
| 주술 호응 불일치   | 내 꿈은 의사가 되고 싶다 → 내 꿈은 의사가 되는 것이다  |
| 목적어-서술어 호응 | 음악과 그림을 듣는다 → 음악을 듣고 그림을 본다         |
| 조사 오류          | 학교를 갔다 → 학교에 갔다(문맥)                        |
| 사동·피동 오류     | 소개시키다 → 소개하다 / 설레이다 → 설레다              |
| 의미 중복          | 역전 앞 → 역전 / 미리 예습 → 예습 / 따뜻한 온정 → 온정 |
| 시제·호응 누락     | 비록 ~지만(호응 누락) → 비록 ~지만 ~다 갖춰            |

> 중복·사동 오류는 명백할 때만. 문체적 강조(의도된 반복)는 보존. 호응이 애매하면 표시만 한다.

## 교정 원칙

1. **규범 근거 우선** — 각 교정은 위 분류(맞/띄/표/외/부/문) 중 하나로 분류 가능해야 한다. 분류 안 되면
   그건 문체 문제이지 오류가 아니다 → 손대지 않는다.
2. **의미·문체 불변** — 오류만 고치고 문장의 뜻·어투·격식은 그대로 둔다.
3. **과교정 금지** — 규정상 허용되는 복수 표기(원칙/허용)는 틀린 게 아니다. 일관성만 맞춘다.
4. **불확실하면 표시** — 사이시옷·띄어쓰기 허용 등 판단이 갈리면 고치지 말고 '확인 요망'으로 표시한다.
5. **일관성** — 한 문서 안에서 표기·부호·띄어쓰기 방식을 통일한다.
