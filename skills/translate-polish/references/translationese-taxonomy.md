# 번역투 분류 (translationese taxonomy)

번역된 한국어 글(번역서·자막·논문·기계번역·MTPE)에서 반복되는 **번역투**를 8대 유형으로 정리한
교정 기준이다. AI 티 분류(`../../humanize/references/ai-tell-taxonomy.md`)와 패턴이 겹치지만, 여기서는
**원천 언어(영어·일본어) 간섭**과 **번역 보편소**의 관점으로 본다. 각 유형은 **정의 · 패턴/예시 ·
심각도 · 보존 주의**로 구성된다. 학술 근거는 [`translationese-research.md`](translationese-research.md).

## 심각도 등급

| 등급   | 의미                                           | 처리                     |
| ------ | ---------------------------------------------- | ------------------------ |
| **강** | 한 번만 나와도 번역투가 뚜렷함                 | 원칙적으로 모두 교정     |
| **중** | 1~2회는 자연스러우나 반복되면 번역체가 굳어짐  | 문서당 3회 이상이면 교정 |
| **약** | 단독으로는 무방하고 다른 패턴과 겹칠 때만 부각 | 중첩 시에만 손봄         |

## 원천 언어 게이트

번역투의 **방향**을 먼저 추정한다(원문이 있으면 확인, 없으면 패턴으로 역추정).

- **영한(EN→KO)**: 무생물 주어, 직역 대명사(it/they → 그것/그들), 관계절 중첩, 수동태, `of` 직역 `~의`.
- **일한(JA→KO)**: 이중 피동·사동, `~に対して`(→ ~에 대해) 과다, `~という` 형식, 한자어 직수입, `~의 경우`.
- 방향이 불명확하면 공통 유형(피동·명사화·명시화)부터 본다. **과교정 금지** — 한국어 정상 표현까지 손대지 않는다.

## 1. 무생물 주어 + 타동사

영어의 무생물 주어 구문(This study shows…)을 직역한 어색한 구조. 한국어는 사람·기관이 주어로 오는 게 자연스럽다.

| 패턴                            | 예시 (티 → 자연)                                                             | 심각도 |
| ------------------------------- | ---------------------------------------------------------------------------- | ------ |
| 무생물 주어 + 보여주다/나타내다 | 이 연구는 ~을 보여준다 → 이 연구로 ~을 알 수 있다 / 이 연구에서 ~이 드러난다 | 중     |
| 무생물 주어 + 허용/요구/가능케  | 이 기능은 사용자가 ~하도록 한다 → 이 기능으로 사용자가 ~할 수 있다           | 중     |
| 추상 주어 + 인과 타동사         | 그 결정은 혼란을 야기했다 → 그 결정으로 혼란이 생겼다                        | 중     |

> **근거:** 이영옥(2001)·최서영(2022)은 무생물 주어 타동사 구문을 영한 번역투의 핵심 유형으로 본다.
> 안미영(2020)은 EN-KO MT에서 이 패턴이 두드러진다고 보고.

**주의:** 의인화가 의도된 문학·광고 문장("도시가 잠들었다")은 보존.

## 2. 피동·이중피동 남용

영어 수동태·일본어 수동의 잔재. 능동으로 바꾸면 대부분 또렷해진다.

| 패턴                    | 예시 (티 → 자연)                                                   | 심각도 |
| ----------------------- | ------------------------------------------------------------------ | ------ |
| 이중 피동               | 되어진다 / 보여진다 / 만들어지게 된다 → 된다 / 보인다 / 만들어진다 | 강     |
| `~에 의해` 수동         | 위원회에 의해 결정되었다 → 위원회가 결정했다                       | 중     |
| `~로 여겨진다/보여진다` | 중요하다고 여겨진다 → 중요하다 / 중요하게 여긴다                   | 중     |

> **근거:** 오경순(2010)·최유숙(2023). 피동·이중피동은 일한·영한 번역투의 대표 유형.

**주의:** 행위자를 의도적으로 감추는 공식·법률·과학 문체의 피동("표본은 4℃에서 보관되었다")은 보존 가능.

## 3. 직역 대명사

영어 인칭·지시 대명사(it, they, this)를 그대로 옮긴 군더더기 대명사. 한국어는 생략이 자연스럽다.

| 패턴                      | 예시 (티 → 자연)                                | 심각도 |
| ------------------------- | ----------------------------------------------- | ------ |
| 문두 `그것은/이것은`      | 그것은 중요하다 → (생략) 중요하다               | 강     |
| 반복 `그들/그녀/그`       | 그들은 모였고 그들은 논의했다 → 모여서 논의했다 | 강     |
| 형식 주어 직역(it ~ that) | 그것은 명백하다, ~라는 것이 → 분명히 ~하다      | 중     |

> **근거:** 김도훈(2009). 직역 대명사는 영한 번역투의 핵심 표지.

**주의:** 인물 인용·대화에서 지시 대상이 모호해지면 대명사 대신 **명사로 복원**(생략하지 말 것).

## 4. 무생물 복수 '-들'

영어 복수 -s를 무생물 명사에 기계적으로 붙인 형태. 한국어는 무생물에 복수 표지를 잘 쓰지 않는다.

| 패턴             | 예시 (티 → 자연)                                                  | 심각도 |
| ---------------- | ----------------------------------------------------------------- | ------ |
| 무생물 + `-들`   | 정보들 · 문제들 · 결과들 · 데이터들 → 정보 · 문제 · 결과 · 데이터 | 중     |
| 추상명사 + `-들` | 가능성들 · 방법들 → 여러 가능성 · 여러 방법                       | 중     |

> **근거:** 김도훈(2009)·김정우(2010). 국어학적으로 한국어 복수 '-들'은 유정물·특정성 조건이 있다(전영철 2015).

**주의:** 사람·동물 등 유정물 복수("학생들", "아이들")는 정상. 셀 수 있는 개별 대상을 강조할 땐 유지.

## 5. 외국어식 조사·후치사 결합

전치사·후치사를 직역한 과잉 조사 결합. 조사를 단순화하면 자연스러워진다.

| 패턴                    | 예시 (티 → 자연)                              | 심각도 |
| ----------------------- | --------------------------------------------- | ------ |
| `~에 대해/~에 관해`     | 환경에 대해 논의한다 → 환경을 논의한다        | 중     |
| `~을 통해/~를 통하여`   | 교육을 통해 성장한다 → 교육으로 성장한다      | 중     |
| `~에서의/~로서의/~에의` | 학교에서의 생활 → 학교 생활                   | 중     |
| `~와 함께`(with 직역)   | 위험과 함께 온다 → 위험을 동반한다 / 위험하다 | 약     |

> **근거:** 최유숙(2023)·김정우(2007). 영어/일본어식 조사 결합은 번역투의 빈출 표지.

**주의:** `~을 통해`가 실제 매개·경로를 뜻하면(터널을 통해) 보존. 도식적 추상 경로일 때만 교정.

## 6. 시제·서법 calque

영어 시제·완료상·서법을 직역한 어색한 종결. 한국어 시제 체계로 환원한다.

| 패턴                      | 예시 (티 → 자연)                                   | 심각도 |
| ------------------------- | -------------------------------------------------- | ------ |
| 완료상 직역 `~해 왔다`    | 노력해 왔다(남발) → 노력했다                       | 약     |
| 진행 직역 `~하고 있는 중` | 검토하고 있는 중이다 → 검토하고 있다 / 검토 중이다 | 약     |
| 가정·조건 calque          | ~였더라면 좋았을 것이다(남발) → 맥락에 맞게 단순화 | 약     |

> **근거:** 최유숙(2023). 시제·서법 calque는 빈도는 낮으나 번역체 인상을 강하게 남긴다.

**주의:** 실제 지속·진행 의미가 있으면 보존. 의미 변질에 특히 주의(시제는 사실 정보).

## 7. 명시화 과잉 (접속어·수식 증식)

번역 보편소 '명시화(explicitation)'. 원문에 없던 접속어·수식·반복을 더해 문장이 늘어진다.

| 패턴                 | 예시 (티 → 자연)                                    | 심각도 |
| -------------------- | --------------------------------------------------- | ------ |
| 불필요한 접속어 삽입 | 그러므로 · 따라서 · 또한 (문두 남발) → 필요한 것만  | 중     |
| 동어반복 수식        | 미리 예측하다 · 함께 협력하다 → 예측하다 · 협력하다 | 중     |
| 잉여 설명구          | ~라고 하는 것은 ~을 의미한다 → ~은 ~이다            | 중     |

> **근거:** Baker(1996) 명시화, 최희경(2016b). 번역문은 비번역문보다 접속·연결이 명시적이고 길어진다.

**주의:** 원문의 논리를 살리는 핵심 접속어(역접·인과)는 남긴다. 0으로 만들지 말 것.

## 8. 명사화·경동사

한자어 명사화와 경동사(light verb) 분리로 문장이 둔해진다. 동사로 되돌리면 또렷해진다.

| 패턴                           | 예시 (티 → 자연)                                    | 심각도 |
| ------------------------------ | --------------------------------------------------- | ------ |
| 경동사 분리                    | 활용을 하다 · 검토를 진행하다 → 활용하다 · 검토하다 | 중     |
| 한자어 대동사 `실시/수행/진행` | 조사를 실시하다 → 조사하다                          | 중     |
| 과도한 명사화                  | ~의 증가를 보였다 → 늘었다                          | 중     |

> **근거:** 명사화·문법적 은유는 정려란(2018)·소지영·주세형(2017). 경동사 간결화는 홍성호(2020)·이오덕·
> 국립국어원 「쉬운 공문서 쓰기 길잡이」 등 실무·규범 처방(학술 1차 문헌은 미발굴, 2026-06 확인).

**주의:** 의미상 명사화가 더 정확한 전문·법률 표현("계약의 해지")은 보존.

## 보존 원칙 (Do-NOT)

번역 윤문은 **번역의 의미를 한 글자도 바꾸지 않는다.** 다음은 어떤 경우에도 보존한다.

- **고유명사·인명·지명·기관명** — 표기 통일은 하되 의미·대상 변경 금지.
- **수치·단위·날짜·통화·환율** — 원문 정보 그대로(환산 임의 추가 금지).
- **직접 인용·대사** — 따옴표 안 문장은 번역투라도 화자 특성이면 보존 판단.
- **전문 용어·정의** — 분야 표준 역어가 있으면 따른다(임의 의역 금지).
- **주장의 방향·강도·인과·수량** — 증가↔감소, 대부분↔일부 등 변질 금지.
- **좌분지(left-branching) 수식** — 한국어의 **정상** 특징. '관형절이 길다'는 이유만으로 손대지 않는다.
  영어 어순을 모사한 **우향 분기·관형격 '의' 반복**일 때만 교정(과교정 금지).
- **원문 대조 시** — 원천 텍스트가 제공되면 의미 누락·왜곡 여부를 우선 확인하되, 번역 자체를 다시 하지는 않는다.
