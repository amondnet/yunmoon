# 번역 윤문 실행 가이드 (translate-polish guide)

`translationese-taxonomy.md`로 탐지한 번역투를 **의미는 보존한 채** 자연스러운 한국어로 고쳐 쓰는
방법과, 끝난 뒤 스스로 점검하는 체크리스트를 담는다. 번역 윤문은 **재번역이 아니다** — 이미 번역된
한국어 문장의 *번역체*만 다듬는다.

## 핵심 원칙

1. **의미 100% 보존** — 번역문이 담은 정보(수치·고유명사·인용·주장)를 바꾸지 않는다. 오역 의심이 보여도
   임의로 고치지 말고 **표시만** 한다(번역 윤문의 일이 아니다 → 사용자에게 알림).
2. **재번역 금지** — 원문이 있어도 처음부터 다시 옮기지 않는다. 탐지된 번역투 구간만 손댄다.
3. **원천 언어 의식** — 영한/일한 간섭 방향을 추정해 해당 유형을 우선 본다(`translationese-taxonomy.md` 게이트).
4. **과교정 금지** — 한국어의 정상 표현(좌분지 수식, 유정물 복수, 매개 의미의 '~을 통해')까지 손대지 않는다.
5. **인공 표현 추가 금지** — 번역투를 없애려다 또 다른 어색한 말투를 만들지 않는다.

## 강도

사용자 지정(`강도: 보수|기본|적극`)에 따라 손대는 범위를 조절한다. 기본값은 **기본**.

| 강도     | 처리 범위                                                      |
| -------- | -------------------------------------------------------------- |
| **보수** | 심각도 **강**만 교정(이중피동·직역 대명사 등). 구조 보존       |
| **기본** | 강 + 반복되는 **중**(문서당 3회 이상) 교정                     |
| **적극** | 강 + 중 전반 + 명시화 과잉·문장 분할까지 적극 (의미 보존 동일) |

## 교정 순서

먼저 군더더기를 덜어내면 문장이 짧아져 뒤 작업이 쉬워진다.

1. **명시화 과잉 덜기**(7) — 원문에 없던 잉여 접속어·동어반복 수식부터 들어낸다.
2. **직역 대명사 정리**(3) — 생략하거나 지시 대상을 명사로 복원.
3. **무생물 주어 풀기**(1) — 사람·기관 주어 또는 부사구로 환원.
4. **피동 → 능동**(2) — 행위자가 분명하면 능동으로.
5. **조사·후치사 단순화**(5) — `~에 대해`·`~을 통해`·`~에서의` 정리.
6. **무생물 복수 '-들' 정리**(4)
7. **명사화·경동사 풀기**(8) — 명사 덩어리를 동사로.
8. **시제·서법 calque 환원**(6) — 한국어 시제 체계로(의미 변질 주의).
9. **관형절 중첩 분할** — 우향 분기 모사만 절을 끊어 분할(좌분지는 보존). 마지막에 전체 리듬을 읽으며 손본다.

## 원문(원천 텍스트) 활용

원문이 함께 제공되면:

- **누락·왜곡 점검** — 번역문이 원문 정보를 빠뜨리거나 왜곡했으면 **표시만** 하고 사용자에게 알린다(직접 재번역 금지).
- **명시화·간섭 확인** — 원문 대비 더해진 접속어·수식(명시화), 원문 구조 모사(간섭)를 직접 식별해 교정 우선순위로 삼는다.

원문이 없으면 결과물만 보고 단순화·규범화(T-universals) 패턴으로 번역투를 역추정한다.

## 자체검증 6항 체크리스트

윤문을 마치면 결과물을 번역문(있으면 원문도)과 대조해 점검한다. 위반 시 해당 edit을 **롤백하고 1회 부분 재작업**한다.

1. **정보 보존** — 수치·고유명사·인용·날짜·단위가 번역문과 100% 같은가?
2. **의미 동등** — 주장 방향·인과·수량·강도가 그대로인가? (재번역으로 의미가 바뀌지 않았는가)
3. **변경률 적정** — 과교정이 아닌가? (대략 30~40% 이내, 50% 초과 시 재검토 — 그쯤이면 재번역 의심)
4. **자연스러움** — 번역체가 가셨는가? 소리 내어 읽어 걸리는 곳이 없는가?
5. **강 패턴 잔존 0건** — 이중피동·직역 대명사 등 '강' 번역투가 남아 있지 않은가?
6. **인공 표현 무추가** — 새로 어색한 말투를 만들지 않았는가? 과교정으로 정상 표현을 망가뜨리지 않았는가?

## 변경률·등급 산정

- **변경률** = 대략적 수정 비율(체감 추정). 50% 초과는 과교정/재번역 신호 → 보수적으로 되돌린다.
- **등급**:

| 등급  | 기준                                                         |
| ----- | ------------------------------------------------------------ |
| **A** | 강 잔존 0, 자체검증 6/6, 변경률 적정, 정보·의미 보존         |
| **B** | 강 잔존 0, 자체검증 5/6 이상, 사소한 중 패턴 일부 잔존       |
| **C** | 강 1건 이하 잔존 또는 변경률 과다(재번역 우려) — 재검토 권장 |
| **D** | 강 다수 잔존 또는 의미 훼손 위험 — 부분 재작업 필요          |

등급 B 이하이거나 의미 보존이 불안하면 사용자에게 한 줄로 알리고 추가 점검(특정 유형만 재실행, 강도 하향)을 제안한다.
